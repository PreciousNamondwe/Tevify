# Tevify

## Running the Expo App

Follow these steps to set up and run the Tevify Expo app.

### Prerequisites

- **Node.js and npm**: Ensure Node.js (v16 or higher) and npm are installed. Verify with:
  ```bash
  node -v
  npm -v
  npm install -g expo-cli
  cd path/to/tevify
  npm install
  npx expo start
